# Introducao a HMTL:

## Tags em HTML:

***Titulos***
- h1
- h2
- h3
- h4
- h5
- h6

***Lista***
- li
- ul

***div tipos***
- class ou id -> Referenciar no css como # ou .

***Parágrafos***
- p

***strong***
- Texto em negrito (semântico)

***imagem***
- img: Para adicionar imagens à página.
Exemplo: <img src="imagem.jpg" alt="Descrição da imagem">.

***Tabelas***

table: Define uma tabela.

tr: Linha da tabela.

td: Célula da tabela.

th: Cabeçalho da tabela.
